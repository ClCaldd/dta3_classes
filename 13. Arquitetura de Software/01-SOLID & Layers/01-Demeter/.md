# Law of Demeter (LoD)
### Lei de Demeter

A "Lei de Demeter" define que um objeto deve saber o minímo possível sober a estrutura interna dos dmeais. Portanto, A contendo um objeto B que contém o objeto C, qualquer acesso do C partindo do A deve ser evitado:

    A.GetB().GetC()

A Lei diz que um método M da classe A só deve acessar:
- Outro métodos de A
- Métodos de objetos passado para ele por parâmetro
- Objetos criados dentro do escopo de M
- De Atributos de A
---

## Demonstração
```cs
public class Usuario
{
    public Empresa Empresa {get;set;}
}

public class Empresa
{
    public Endereco Endereco {get;set;}
}

public class Endereco
{
    public Cidade Cidade {get;set;}
}
```
---
Descumprindo a lei:
```cs
public class Usuario
{
    public Empresa Empresa {get;set;}
    public Cidade GetCidade()
    {
        var cidade = this.Empresa.Endereco.Cidade
        return cidade;
    }
}
```
---
Cumprindo a lei:
```cs
public class Usuario
{
    public Empresa Empresa {get;set;}
    public Cidade GetCidade()
    {
        var cidade = this.Empresa.GetCidade();
        return cidade;
    }
}

public class Empresa
{
    public Endereco Endereco {get;set;}
    public Cidade GetCidade()
    {
        var cidade = this.Endereco.Cidade;
        return cidade;
    }
}

```
