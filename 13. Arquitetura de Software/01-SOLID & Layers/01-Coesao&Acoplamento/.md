# Coesao e Acoplamento

### Coesão
Coesão é quão próximas as funcionalidades estão entre si, se faz sentido todas elas estarem na mesma classe.
Pode ser confundido com o Principio de Responsabilidade Unica (SRP), eles são próximos mas o SRP é uma regra 
definida para conseguir alcançar uma alta coesão. Basicamente coesão é a métrica apenas.

---


### Acoplamento
Acoplamento é o quao dependente uma classe é de outras, por exemplo alto acoplamento seria:
```cs
public class Usuario
{
    private CalculadoraCientifica calculadora = new();

    public int SomarNumeros(int a, int b)
    {
        return calculadora.Soma(a, b);
    }
}
```
Usuario está fortemente acoplado a classe CalculadoraCientifica, porque:
instancia diretamente a classe concreta (CalculadoraCientifica)
não permite trocar implementação 
mudanças em CalculadoraCientifica podem quebrar Usuario

Mesmo que CalculadoraCientifica implemente de uma interface "ICalculadora", é impossível para o usuário utilizar outra calculadora.

---

E agora um exemplo de baixo acoplamento:
```cs
public class Usuario(ICalculadora calculadora)
{
    ICalculadora Calculadora = calculadora;
    public int SomarNumeros(int a, int b)
    {
        return Calculadora.Soma(a, b);
    }
}
```

# Objetivo
No mundo ideal queremos alcançar alta coesão e baixo acoplamento.

Vale lembrar que não é só definir o tipo como o da interface, pois também é possível ter alto acoplamento mesmo usando interfaces, por exemplo:

```cs
public class Usuario
{
    private readonly ICalculadora calculadora = new CalculadoraCientifica();

    public int SomarNumeros(int a, int b)
    {
        return calculadora.Soma(a, b);
    }
}
```
Apesar de depender de ICalculadora, a classe ainda está acoplada à implementação CalculadoraCientifica, pois ela é criada diretamente dentro da classe

## Relação com SOLID 
**S**RP - Melhora a coesão (SRP é causa, coesão é efeito)

**O**CP - Reduz o acoplamento (diretamente)

**I**SP - Reduz o acoplamento 

**L**SP - Reduz o acoplamento 

**D**IP - Reduz o acoplamento (diretamente)
